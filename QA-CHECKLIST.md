# Allround English QA 체크리스트

> 코드 수정/배포 후 반드시 아래 항목을 브라우저에서 직접 확인합니다.
> 체크 완료 시 [ ] → [x] 로 변경. 문제 발견 시 해당 줄에 메모.
> **예상 소요: 15~20분**

---

## A. 회원가입 / 로그인 (auth.html)

### A-1. 회원가입
- [ ] 이메일/비밀번호로 가입 → 성공 후 courses.html 이동
- [ ] 필수 입력 누락 시 오류 메시지 표시 (이름, 이메일, 비번, 보호자)
- [ ] 비밀번호 8자 미만 → 오류 메시지
- [ ] 이미 가입된 이메일 → "이미 가입된 이메일" 메시지
- [ ] 보호자 이름/연락처 입력 확인
- [ ] 주소 검색 (다음 우편번호) 동작 확인

### A-2. 로그인
- [ ] 정상 로그인 → courses.html 이동
- [ ] 잘못된 비밀번호 → 오류 메시지
- [ ] 로그인 후 헤더에 MY PAGE / LOGOUT 표시
- [ ] 관리자 계정 로그인 → ADMIN 링크 표시 (노란색)

### A-3. 비밀번호/이메일 찾기
- [ ] 비밀번호 찾기 → 이메일 발송 메시지
- [ ] 이메일 찾기 (이름+연락처) → 마스킹된 이메일 표시

### A-4. 세션 관리
- [ ] 다른 브라우저에서 동시 로그인 → 3대 초과 시 가장 오래된 세션 해제
- [ ] 로그아웃 → 세션 정리 + courses.html 이동

---

## B. 강의 목록 / 상세 (courses.html, course-detail.html)

### B-1. 강의 목록
- [ ] 페이지 로딩 → 강의 카드 표시
- [ ] 비공개(private) 강의 → 권한 없는 학생에게 미표시
- [ ] 미발행(unpublished) 강의 → 일반 학생 미표시, 관리자만 표시
- [ ] 강의 카드 클릭 → course-detail.html 이동

### B-2. 강의 상세
- [ ] 강의 정보 표시 (제목, 설명, 레슨 수, 총 시간)
- [ ] 레슨 목록 순서 (order_num) 정상
- [ ] **미구매 상태**: 가격 표시 + "수강 신청" 버튼
- [ ] **구매 완료 상태**: 진도 바 표시 + 레슨 클릭 가능
- [ ] 무료 강의 → "무료 수강" 버튼 → 즉시 수강 등록
- [ ] 비로그인 시 → auth.html 이동 (redirect 파라미터 포함)

## C. 강의 수강 (lesson.html) ★ 가장 중요

### C-1. 영상 재생
- [ ] Vimeo 영상 로딩 → 커스텀 플레이어 표시
- [ ] 재생/일시정지 버튼 동작
- [ ] 볼륨 조절 동작
- [ ] 전체화면 동작
- [ ] 영상 시간 표시 (현재/전체)

### C-2. 진도 추적 ★★★
- [ ] 영상 재생 시 "시청 진도율" 실시간 증가
- [ ] 일시정지 후 재개 → 진도율 계속 증가 (멈추지 않음)
- [ ] **탭 전환 후 복귀 → 진도율이 계속 올라가는지 확인** ★
- [ ] **영상 버퍼링 발생 시 → 진도율이 멈추지 않는지 확인** ★
- [ ] 페이지 새로고침 → 이전 진도 유지 (DB에서 불러옴)
- [ ] 다른 레슨으로 이동 후 → 돌아왔을 때 진도 유지
- [ ] 95% 이상 시청 → "완료" 처리 (과제 섹션 표시)
- [ ] 영상 끝까지 시청 → 과제 모달 자동 표시

### C-3. 이어보기
- [ ] 영상 중간에 페이지 이탈 → 재진입 시 이전 위치에서 이어보기
- [ ] 모바일에서 페이지 이탈 → 다시 열면 이전 위치 복원

### C-4. 과제
- [ ] 95% 미만 시청 시 → 과제 섹션 숨김
- [ ] 퀴즈 모달 → 답변 선택 → 결과 표시 → 완료 체크
- [ ] 노트 과제 → 체크 → 완료 표시
- [ ] 블로그 과제 → 체크 → 완료 표시
- [ ] 이미지 과제 → 파일 선택 → 업로드 → 완료 표시
- [ ] 과제 완료 상태가 새로고침 후에도 유지

### C-5. 레슨 탐색
- [ ] 사이드바 커리큘럼 → 다른 레슨 클릭 → 이동
- [ ] 이전/다음 버튼 동작
- [ ] 첫 번째 레슨 → "이전" 버튼 비활성
- [ ] 마지막 레슨 → "다음" 버튼 비활성

### C-6. QnA
- [ ] 플로팅 버튼 → QnA 모달 열기
- [ ] 질문 작성 → 전송 → 목록에 표시

---

## D. 마이페이지 (mypage.html)

### D-1. 수강 강의
- [ ] 구매한 강의 목록 표시
- [ ] 각 강의 진도율(%) 정상 표시
- [ ] 강의 카드 클릭 → course-detail.html 이동

### D-2. 주문/배송
- [ ] 주문 내역 표시 (날짜, 강의명, 금액, 상태)
- [ ] 배송 상태 표시 (해당 시)

### D-3. 프로필 설정
- [ ] 현재 정보 로딩 (이름, 이메일, 연락처, 주소)
- [ ] 정보 수정 → 저장 → 새로고침 후 유지
- [ ] 보호자 정보 수정/저장

### D-4. 내 기기
- [ ] 접속 중인 기기 목록 표시
- [ ] 다른 기기 로그아웃 버튼 동작

### D-5. 쿠폰
- [ ] 보유 쿠폰 표시 (할인율, 만료일)
- [ ] 사용 완료/만료 쿠폰 분리 표시

### D-6. 내 질문
- [ ] 질문 목록 표시
- [ ] 답변 완료/대기중 상태 표시

---

## E. 결제 (payment.html → success/fail)

### E-1. 결제 페이지
- [ ] 강의 정보 표시 (이름, 가격)
- [ ] 결제 항목 선택 (강의비/교재비)
- [ ] 쿠폰 코드 입력 → 할인 적용
- [ ] 결제 총액 표시
- [ ] 카드/가상계좌 선택

### E-2. 결제 진행
- [ ] "결제하기" → Toss 결제창 표시
- [ ] 결제 성공 → payment-success.html 이동 → "수강 시작" 버튼
- [ ] 결제 실패/취소 → payment-fail.html 이동 → "다시 결제" 버튼
- [ ] 결제 완료 후 → 강의 접근 가능 확인

## F. 수강 리포트 (report.html)

- [ ] 강의 완료 후 리포트 페이지 접근
- [ ] 진도율, 과제율, 시청률 차트 표시
- [ ] 수강증 공유 링크(public_id) 동작
- [ ] 쿠폰 발급 버튼 → 정상 발급
- [ ] 추천 강의 표시

## G. 관리자 (admin.html)

### G-1. 접근 제어
- [ ] 관리자 아닌 계정 → 접근 차단 (courses.html로 이동)
- [ ] 관리자 계정 → 정상 접근

### G-2. 강의 관리
- [ ] 강의 목록 표시
- [ ] 강의 생성/수정/삭제
- [ ] 레슨 추가/수정/삭제
- [ ] 비공개 강의 접근 권한 관리

### G-3. 회원 관리
- [ ] 학생 목록 표시 (이름, 이메일, 연락처)
- [ ] 학생 검색 동작

### G-4. 과제 관리
- [ ] 학생별 과제 현황 표시
- [ ] **학생 진도율(%) 관리자 화면에서 정상 표시** ★
- [ ] **관리자 계정이 과제/진도 데이터에 표시되지 않는지 확인** ★
- [ ] 이미지 과제 채점 → 점수 입력 → 저장
- [ ] 채점 저장 후 → 알림톡 발송 (학생+학부모 번호)
- [ ] Courses 탭 → 강의별 학생 진도 테이블에 관리자 미표시

### G-5. 주문/결제 관리
- [ ] 주문 목록 표시
- [ ] 배송 상태 변경

### G-6. QnA 관리
- [ ] 질문 목록 표시
- [ ] 답변 작성 → 전송

### G-7. 알림톡 (수동 트리거) ★
- [ ] 알림톡 발송 내역 탭 → 로그 표시 (테스트 유저 제외 확인)
- [ ] 발송 상태 (성공/실패) 필터링 동작
- [ ] **채점 완료(grading_complete) 시 알림톡 발송** ★★★
  - [ ] 학생 연락처로 발송
  - [ ] 학부모 연락처로 발송
  - [ ] notification_logs 테이블에 기록
  - [ ] 변수: 학생명, 과제명, 점수 (14자 초과 시 잘림)
- [ ] **강의 완료(course_complete) 시 알림톡 발송** ★★
  - [ ] 전체 레슨 완료 시 자동 트리거
  - [ ] 변수: 학생명, 과정명, 진도율, 숙제완료율
- [ ] 알림 비활성화한 학생에게는 미발송 확인
- [ ] 연락처 없는 학생 → 에러 없이 건너뜀 확인

### G-8. 알림톡 자동 발송 (pg_cron, 매일 오전 9시)
- [ ] Supabase Edge Function `send-nhn-alimtalk` 배포 상태
- [ ] Supabase Edge Function `auto-send-notifications` 배포 상태
- [ ] pg_cron job 등록 확인 (`SELECT * FROM cron.job`)
- [ ] pg_cron URL이 실제 프로젝트 URL인지 확인 (플레이스홀더 아닌지)
- [ ] **과제 미제출(assignment_reminder) 알림** ★★
  - [ ] 레슨 완료 + 24시간 경과 + 퀴즈 등록된 레슨만 대상
  - [ ] 과제가 없는 레슨은 제외
  - [ ] 24시간 내 중복 발송 방지
  - [ ] 변수: 학생명, 과정명, 강의제목 (14자 잘림 처리)
- [ ] **미수강(inactive_reminder) 알림** ★★
  - [ ] 구매 후 7일 미수강 학생 대상
  - [ ] 7일 내 중복 발송 방지
  - [ ] 변수: 학생명, 과정명, 경과일수
- [ ] 관리자 계정 알림 대상 제외 확인
- [ ] NHN 템플릿 4개 승인 상태 확인 (NHN Console)
- [ ] NHN 템플릿 변수값 14자 제한 처리 확인
