=================================================================
NHN Cloud 알림톡 템플릿 등록 가이드
=================================================================

이 파일은 NHN Cloud Console에서 알림톡 템플릿을 등록할 때 사용할 템플릿 예시입니다.

-----------------------------------------------------------------
템플릿 1: 과제 미제출 알림
-----------------------------------------------------------------

템플릿 코드: assignment_reminder
템플릿 이름: 과제 제출 알림
카테고리: 교육/학습

[템플릿 내용]
----------------------------------------
[올라운드] 과제 제출 알림

#{name}님, 안녕하세요!

#{courseName} - #{lessonTitle} 강의를 수강하신 후 24시간이 지났습니다.

과제를 아직 제출하지 않으셨네요! 🎯
학습 효과를 극대화하기 위해 지금 바로 과제를 제출해주세요.

👉 과제 제출하기
https://allround.co.kr/courses

궁금한 점이 있으시면 언제든 문의해주세요!

올라운드 영어 드림
----------------------------------------

[템플릿 변수]
- name: 학생 이름
- courseName: 강의 이름
- lessonTitle: 레슨 제목

[버튼 설정 (선택사항)]
- 버튼명: 과제 제출하기
- 버튼 타입: 웹 링크
- 버튼 링크: https://allround.co.kr/courses


=================================================================
템플릿 2: 채점 완료 알림
=================================================================

템플릿 코드: grading_complete
템플릿 이름: 채점 완료 알림
카테고리: 교육/학습

[템플릿 내용]
----------------------------------------
[올라운드] 채점 완료 알림

#{학생명}님, 안녕하세요!

#{과제명} 과제 채점이 완료되었습니다.

점수: #{점수}점
피드백: #{피드백}

자세한 피드백은 올라운드 사이트에서 확인해주세요.

https://allround.co.kr/lesson

올라운드 영어 드림
----------------------------------------

[템플릿 변수]
- 학생명: 학생 이름
- 과제명: 레슨 번호 (예: 1강)
- 점수: 채점 점수
- 피드백: 선생님 피드백

[버튼 설정 (선택사항)]
- 버튼명: 결과 확인하기
- 버튼 타입: 웹 링크
- 버튼 링크: https://allround.co.kr/lesson


=================================================================
템플릿 3: 장기 미수강 알림
=================================================================

템플릿 코드: inactive_reminder
템플릿 이름: 학습 독려 알림
카테고리: 교육/학습

[템플릿 내용]
----------------------------------------
[올라운드] 학습 독려 알림

#{name}님, 안녕하세요!

#{courseName} 강의를 #{days}일 동안 수강하지 않으셨어요. 📚

영어 실력 향상을 위해서는 꾸준한 학습이 중요합니다!
오늘 잠깐이라도 강의를 들어보는 건 어떨까요?

👉 강의 이어보기
https://allround.co.kr/courses

함께 목표를 달성해봐요! 💪

올라운드 영어 드림
----------------------------------------

[템플릿 변수]
- name: 학생 이름
- courseName: 강의 이름
- days: 경과 일수

[버튼 설정 (선택사항)]
- 버튼명: 강의 이어보기
- 버튼 타입: 웹 링크
- 버튼 링크: https://allround.co.kr/courses


=================================================================
NHN Cloud Console 등록 절차
=================================================================

1. NHN Cloud Console 접속
   https://console.toast.com

2. KakaoTalk Bizmessage → 알림톡 관리로 이동

3. 템플릿 관리 → 템플릿 등록 클릭

4. 위 템플릿 내용을 복사하여 등록

5. 템플릿 검수 요청 (1~2일 소요)

6. 승인 후 템플릿 코드 확인 및 시스템에 적용


=================================================================
주의사항
=================================================================

✅ 템플릿 변수는 반드시 #{변수명} 형식으로 작성
✅ 강조 표시를 위한 이모지 사용 가능
✅ 템플릿 승인까지 1~2일 소요
✅ 광고성 내용은 반드시 [광고] 표기 필요
✅ 수신거부 방법 안내 필수 (080 번호 등)

❌ 개인정보 과다 수집 금지
❌ 허위/과장 광고 금지
❌ 금융/의료 등 규제 업종은 별도 승인 필요
